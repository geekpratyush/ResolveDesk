System Overview

ResolveDesk is a microservices-based support ticketing system with 3 .NET backend services behind a YARP reverse proxy gateway and an Angular 17 SPA frontend.

Technology Stack

Layer Technology Purpose Key Files
Frontend Angular 17 (Standalone Components, Signals) Reactive SPA dashboard with role-based views src/ResolveDesk.UI/
API Gateway YARP (Yet Another Reverse Proxy) Single entry point routing to microservices src/ResolveDesk.Gateway/appsettings.json
Identity Service ASP.NET Core Identity + JWT Bearer User registration, login, role management src/ResolveDesk.Services.Identity/
Ticket Service ASP.NET Core Web API + MassTransit CRUD tickets, responses, status transitions, event publishing src/ResolveDesk.Services.TicketCore/
Database EF Core 8 — SQLite (local) / SQL Server (prod) Dual-provider strategy via connection string detection Program.cs in each service
Message Bus MassTransit InMemory Transport Decoupled event publishing without external infrastructure Consumers/ folder in TicketCore
Authentication JWT Bearer Tokens (HS256) Stateless API authentication with role claims AuthController.cs, auth.interceptor.ts
